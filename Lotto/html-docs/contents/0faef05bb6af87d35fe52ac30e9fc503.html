Description

유스케이스 명 - 로또 번호 생성

액터 - 사용자

사전 조건 - 사용자는 로또 번호 생성 프로그램을 실행하여 메인 화면을 활성화 해야 한다.

사후 조건 - 생성된 번호는 재생성 및 저장을 할 수 있습니다.

기본 흐름 - 1. 사용자를 로또 번호 생 성 프로그램을 실행한다, - 2. 자동 또는 반자동을 선택하여 로또 번호가 생성되기 위한 방법을 지정한다. - 3. 생성 버튼을 클릭하여 로또 번호를 생성한 다.

대안 흐름 A. 자동을 선택한 경우 - 1. 생성 버 튼을 클릭하여 로또 번호 6개를 생성한다. B. 반자동을 선택한 경우 - 1. 사용자가 번호를 선택할 수 있는 체크박스를 활성화하여 번호를 선택하도록 한다. - 2. 생성 버튼을 클릭하여 선택한 번호를 제외한 나머지 번호가 생성되도록 한다.
