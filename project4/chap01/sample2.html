<br> : 브라우저에서 개행이 될    수 있게 한다.
<hr> : 브라우저에서 가로 줄을 생성한다.
<strong> : 강조의 의미를 가지는 태그
<b> : 글자를 굵게 나타내기 위해 사용하는 태그 <mark> : 글자를 노란색 배경으로 하이라이팅 하는 태그 <u> : 글자에 밑줄을 긋기위해 사용하는 태그
<s> : 글자에 취소선을 긋기위해 사용하는 태그
<em> : 강조의 의미를 가지는 태그
<i> : 글자를 기울임 형태로 나타내기 위해 사용하는 태그
<small> : 글자를 크기를 작게 나타내기 위해 사용하는 태그
<blockquote> : 다른 사이트의 글을 인용하여 사용한 경우 쓰이는 태그(개행을 내포함)
<q> : 다른 사이트의 글을 인용하여 사용한 경우 쓰이는 태그 ""를 표시하고 개행을 내포하지 않음
<sup> : 위 첨자 표기를 위한 태그
<sub> : 아래 첨자 표기를 위한 태그
<abbr> : 두문자어 또는 약어를 사용한 글자에 적용하며, 해당 문자에 툴팁 형태로 값이 나온다.
<cite> : 저막물의 제목/출처 표기를 위해 사용하는 태그
<code> : 소스코드를 담아서 사용할 때 쓰이는 태그
<kbd> : 키보드 입력과 같은 내용을 담아서 사용할 때 쓰이는 태그
예) 저장 단축키 : Ctrl + S
pre 태그안에 작성된 글은 개행에 대한 처리가 입력하 내용대로
        출력되어서 보여진다.
        그래서 소스코드 입력과 같은
        여러줄에 걸쳐 입력된 내용은 pre 태그에 넣어 출력하면
        소스코드의 내용 그대로 나오게 된다.
